sqli-labs环境搭建

发表于 2018-01-23 | 分类于 web

最近想学一学web方面的基本知识，先从SQL注入学起。sqli-labs以闯关的方式来介绍了SQL注入的许多技巧。在闯关之前需要先搭建sqli-labs环境，这里搭建环境时主要参考了这条链接。

遇到的问题

无法安装php5

  1. sqli-labs中使用的是php5，在连接数据库时使用的是mysql_connect函数，这些函数在php7中已经废弃了。同时，在Ubuntu 16.04中，安装php时默认的都是php7，安装php5时会提示找不到源的错误。这里解决的方式为：

    sudo add-apt-repository ppa:ondrej/php    
sudo apt-get update
sudo apt-get install php5.6

连接数据库失败

  1. 进行上述安装之后在配置数据库时仍然有问题，仍旧卡死在这个界面：

  1. 看到参考链接里面说了需要先测试数据库是否能正常连接，于是使用提供的脚本：

    <?php
$con = mysql_connect("127.0.0.1","root","123456") or die(mysql_error())
echo "hello";
if ($con) {
        echo "OK!";
}else {
       echo "error".mysqli_error();
}
?>

    测试后发现hello根本就没有输出，而且也没有显示出来mysql_error()的值。也就是mysql_connect()函数根本就没有被执行。

  2. 后来查看Apache的日志文件发现如下错误：


    函数根本就没有定义，发现问题后搜索一下就知道是因为mysql安装的不完整。

  3. 安装mysql

    sudo apt-get install mysql-client
sudo apt-get install php5.6-mysql
